This package contains operations for spam text augmentation, including

- [insert junk characters (插入无意义字符)](./aug_insert_junk_chars.py)
- [replace characters with glyph-similar ones (形近字替换)](./aug_glyph.py)
- [replace characters with phonetically similar ones (音近字替换)](./aug_phonetic.py)
- [transform simplified chinese to traditional chinese (简体转繁体)](./aug_sim_to_trad.py)
- [transform chinese to Huoxing language (文字转火星文)](./aug_huoxing_lang.py)
- [replace characters with Pinyin (文字转拼音)](./aug_to_pinyin.py)
- [replace characters with emoji (文字转emoji)](./aug_emoji.py)
